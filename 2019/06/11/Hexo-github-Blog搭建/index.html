建站指南

  • 全步骤指导，跟着做就行：https://zhuanlan.zhihu.com/p/24019875

  • 主题可以在官方库里面找自己喜欢的

    • 更换主题时在Hexo目录下执行 git clone ...，这一条基本都可以在每个主题的github页面找到
    • 记得同时更改_config.yml中的主题名称，如theme: yilia

  • 个人比较喜欢的几个简约主题

    • sky
    • nayo
    • dxx

Tips

  • 基于git，若自己建了文件出现文件不同步的问题，可以选择强行同一master，直接覆盖远程仓库的代码

    git push -uf origin master

  • Hexo基本语法

    • 新建文章并发布

      • 进入你的博客目录，在 /source/_posts 文件夹下直接建立一个.md文件

        右键选择 Git Bash Here 进入命令窗口，输入hexo new "新建文章名称" #新建文章

  • .md文件头格式

blog_site
