null – 赋值为空 VS undefined – 定义了但还未赋值


JS 拥有 type coercion 特性，字符，数字，null(0) 可以一定程度混合运算，混合失败会返回 NaN

如果不需要强制转换，使用 === 或 !===


|| 会返回左边第一个 true 的东西 (false – null/“”/0/NaN)，

相反，&& 返回左边第一个 false 的东西

  2. var 支持同 scope 重复定义，let 不行
